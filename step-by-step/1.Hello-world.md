- Hello World!를 출력하시오.  

## 현식의 코드  
```java
  public class Main {

    public static void main(String[] args) {
        System.out.println("Hello World!");
    }

}

```  

## 현영의 코드
```java
public class hello {
    public static void main(String[] args) {
        System.out.println("Hello world!");
    }
}
```

## 찬구의 코드
```java
class Main {
  public static void main(String[] args) {
    System.out.print("Hello World!");
  }
}
```
